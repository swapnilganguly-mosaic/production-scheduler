Score Formula  ▶
Score = Urgency x Strategic Weight x Material Feasibility
Urgency = 1 / FG DOC  |  FG DOC (days) = FG Stock (units) / DRR (units/day)
Material Feasibility = min(RM DOC, PM DOC) / Target RM DOC  [capped at 1]
RM DOC = min over all RM materials of [ RM Stock (kg) / (RM% x Pack Size kg x DRR) ]  — bottleneck material
PM DOC = min over all PM materials of [ PM Stock (units) / (PM Qty/unit x DRR) ]  — bottleneck material
DRR = Jun Units/day (embedded)  |  RM/PM Stock = live sync from MRP Tool  |  FG Stock = SIH + SIT (manual upload)  |  Target Run QTY = DRR x 13
